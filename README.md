# PMSL

## Py Minecraft Server Launcher

PMSL 全称 Py Minecraft Server Launcher. 
是一个完全使用Python编写的Minecraft游戏服务器快捷创建/启动/配置程序.
此程序完全免费，禁止私自商用.

## 此程序的功能

- 在程序内快捷创建服务器
- 通过程序启动已有服务器
- 在程序内快捷修改服务器设置文件

## 此程序的版本

目前此程序已更新至   **V 1.0.0** .
